## 🎮 How Missions Work

Each mission:
1. Starts with a **business request** (email or message from a stakeholder)
2. Explains **why** this SQL concept exists in the real world
3. Teaches the **SQL syntax** with working examples
4. Provides **exercises** to practice
5. Ends with a **quiz** and **challenge**
6. Awards **XP points**
